# Apple Fold

A personal frontend practice project inspired by modern Apple-style UI design.
The goal of this project is to practice layout, animations, and interactive
JavaScript features.

## Disclaimer

This repository contains a personal frontend practice project.
It was created solely for educational purposes to experiment with layout, styling, and JavaScript behavior.

This project is not an official website and is not affiliated with, endorsed by, or connected to Apple Inc. or any other company.

## Features

- Apple-style search overlay with smooth open/close behavior
- Keyboard interaction (Escape key to close search)
- Translucent “frosted glass” UI elements
- Responsive layout
- Smooth CSS transitions and animations

## Tech Stack

- HTML
- CSS
- JavaScript (Vanilla)

## Purpose

This project exists purely for learning and experimentation.
It is not meant for production use or public deployment as an official site.

## Status

Ongoing — features and UI refinements are added as part of the learning process.
